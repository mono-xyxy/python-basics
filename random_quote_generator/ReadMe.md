# 🎲 Random Quote Generator (Python)

A simple Python project that generates random motivational quotes each time the program runs.

## 📌 Features

* Displays a random quote every time you run the program
* Beginner-friendly Python project
* Can be expanded with APIs, GUI, or a web app
* Lightweight and easy to customize

## 💻 Example Code

```python
import random

quotes = [
    "The best way to get started is to quit talking and begin doing.",
    "Success is not final, failure is not fatal: it is the courage to continue that counts.",
    "Dream big and dare to fail.",
    "Do something today that your future self will thank you for.",
    "Don't watch the clock; do what it does. Keep going."
]

print("\n💡 Random Quote:\n")
print(random.choice(quotes))
```
